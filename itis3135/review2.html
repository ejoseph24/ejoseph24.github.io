Review 2 - Goliber, Kialey

Overview: Excellent page showcasing Dylan's tutoring services and expertise! The page looks professional but not bland, and was intuitive to navigate. Everything was organized effectively, so it was easy to read up on the work and research Dylan has done.

  1. The submission led to the correct page.
  2. All folder names use correct/conventional naming.
  3. Design:
    1. The font style and size are fairly easy to read. The font style and contrast is great, but it might be best to find a bolder version of the font because sometimes it only appears 2-3 pixels wide, making it a bit difficult to read.
    2. The page does use a .css file for styling.
    3. CRAP Principles:
      • Contrast: Elements stand out and draw users to certain pages. I like how the "Testimonials" button in the header is a muted pink (drawing the user to see the people Dylan has helped). I would recommend maybe finding a brighter version of these colors though, as these more muted colors might make the site a bit faded.
      • Repetition: All pages maintain the same font style, header style, and shapes (such as for images or div boxes) for each page.
      • Alignment: All items are center aligned and look naturally placed on the page.
      • Proximity: Elements are grouped and spaced out well, especially on pages such as "Tutoring Services" and "Contact Dylan". The flex gallery looks great, and I like the interactivity of the images getting larger when hovered
  4. Page has within it:
    1. Header: The header for each page looks great.
    2. Main: From what I saw in the code for each page, every page has a main.
    3. Footer: All pages have a minimalistic footer that blends in with the rest of the page (though the link for "Kialey Digital Solutions" gives a 404 error).
    4. Nav Bar: The nav bar is clear to find and intuitive to use.
  5. Header does have the site/brand header with the text in an h1. I don't see a logo, but since this is a tutoring service it's not nessecary.
  6. The main does start with the name of the page as an h2 on almost every page.
  7. Footer:
    1. The footer is clean and minimalistic. If you'd like, you could link Dylan's professional/business socials at the end and include the respective logos for that (LinkedIn, Facebook, etc.)
    2. No HTML or CSS validation links present.
    3. The link to the "Kialey Digital Solutions" does not work.
  8. Personal Recommendations: As previously mentioned in the review, here are some suggestions I would give before you submit your final project (though it looks very close to being complete).
    • Perhaps edit the colors to a brighter equivalent. Right now, it seems a bit dull in color, but a quick change on the brightness will make it pop out more.
    • It might be helpful for Contrast to pick a bolder version of your current font. Most of the paragraph text appears only 2-3 pixels wide on conventional laptop screens, leading to readability issues.
    • Remember to link your HTML/CSS validators in the footer. I think it was a requirement in the project.
    • When the contact form is submitted, a 404 error showed up. Since the form doesen't have to actually collect respnses, just make sure the submission is void once a user hits "Submit".
    • The link to your Digital Solutions page does not work at the moment, so it might just need to be correctly linked.
